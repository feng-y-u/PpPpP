# 测试页设计计划

## 设计方向
- 日系简约风格，参考 blog.yuki.sh 的审美
- 白色/浅色背景，干净排版，大量留白
- 克制、内容优先

## 设计选择
| 项目 | 选择 | 理由 |
|------|------|------|
| 背景 | #f5f5f0 （温暖米白） | 比纯白更柔和，日系常见 |
| 主文字 | #2b2b2b | 柔和黑，不刺眼 |
| 强调色 | #d4436b （珊瑚红） | 日系常用点缀色，克制 |
| 字体 | system-ui, Hiragino Sans, sans-serif | 日系+西文兼容 |
| 排版 | 居中，最大宽度 640px | 阅读舒适，留白充分 |
| 装饰 | 水平细线 + 顶部小圆点标记 | 日系博客经典细节 |

## 内容结构
1. 顶部: 站点名 `flowyu.xyz`（小字）
2. 大标题: `你好，风予`（日式问候风格）
3. 副标题: 简短说明这是测试页 / 域名验证
4. 分割线
5. 服务器信息: IP、时间线状排列
6. 底部: 版权小字
